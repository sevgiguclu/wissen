div, division kelimeisnin kısaltması olarak HTML de etiket olarak kullanılır. Bİr HTML sayfayı tasarımsal alanlara böler

Bazı Özellikleri

  1. ID : Html sayfadaki kullanılan elementin kimliğini belirler...
  2. Width : Bir elementin genişliğini belirler..
  3. Height : Bir elementin yüksekliğini belirler...
  4. margin-left : içeriğin sol tarafındaki çerçeveye olan uzaklığını belirler
  5. margin-right : içeriğin elementin sağ tarafındaki çerçevey olan uzaklığını belirler...
  6. float : ilgili tasarımsal alanın yani div in hangi yöne dayalı olarak tasarıma dahil olacağını belriler(left,right, auto)
