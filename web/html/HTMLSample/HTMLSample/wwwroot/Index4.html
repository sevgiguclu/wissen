Verileri tablo formatında göstermek için kullanılan html element table etiketidir. Tabloda bir satır için tr etiketi sutun için de td etiketi kullanılır..

Tablonun bazı özellikleri

  1. Width : Tablonun genişliği için kullanılır...
  2. border : Tablodaki satır ve sutunların çerçevesinin boyutunu temsil eder...
  3. colspan : Kolonları tek bir kolon haline getirmek için kullanılır. Kolonları merge yapar
  4. rowspan : Satırları tek bir satır haline getirmek kullanılır. Satırları merge yapar
  5. cellspacing : Kolonların birbirleri ile arasındaki mesafeyi temsil eder
  6. cellpadding : Kolon içerisindeki verinin dış çerçeveye olan uzaklığını temsil eder
